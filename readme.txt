
Внимание, при переносе базы из SQL функция проверки кода на все сохранения из нее работать НЕ БУДЕТ
Проверка происходит при загрузке сохранения на сервер, а если вы загрузите 16000 сохранений из другой SQL базы то серверу будет плохо

Перед началом работы убедитесь что вы правильно настроили config.py
Установили и запустили postgres
Установили и запустили redis (можете использовать другой брокер)
Запустили postgresMarkup (!!! он заменит таблицы с названием codes и similarities на свои, стерев из них информацию !!!)
Перенесли вашу базу SQL при помощи converter(если есть)

Чтобы перенести SQL базу вам нужно положить ее в папку где находится код и этот readme
Переименовать ее в database.sqlite
И запустить converter.py
!!! Убедитесь что вы запустили postgres и настроили config.py !!!

Затем запустите две команды ОТДЕЛЬНО, в двух разных терминалах
celery -A celeryApp worker -l info -P eventlet
flask --app app run

Если пишет команда не найдена, допишите в начале каждой из команд python -m 
или python3 -m (смотря какой python у вас установлен)
